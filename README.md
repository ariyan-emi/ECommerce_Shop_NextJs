## if Need run on your machine:

Clone project
```
git clone https://github.com/ariyan-emi/ECommerce_Shop_NextJs.git
```
go to project
```
cd ECommerce_Shop_NextJs
```
install npm
```
npm install
```
start project
```
npm start
```
